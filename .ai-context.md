# AI Context: php-jobspy

## Project Overview
`php-jobspy` is a standalone, decoupled PHP library built to achieve feature parity with the `python-jobspy` project. It scrapes job boards (Indeed, ZipRecruiter, Glassdoor, LinkedIn) without relying on official APIs.

## Current State & Roadmap
- We use `guzzlehttp/guzzle` for all HTTP requests.
- **Current Focus**: Establishing a baseline architecture with a scrape-friendly provider (e.g. Indeed, ZipRecruiter) before tackling highly-restricted boards.
- **LinkedInScraper** is partially implemented but paused due to aggressive auth-walls.
- The roadmap and missing features are strictly tracked in `ROADMAP.md` and the output schema in `SPEC.md`. Always check these files before implementing new features.

## Architectural Rules
- **No Native cURL:** Use Guzzle for all network requests. Enforce strict HTTP timeouts.
- **Graceful Failure:** Scrapers must catch 429/999 AuthWall HTTP errors and return an empty array rather than throwing fatal exceptions that crash the parent pipeline.
- **Data Normalization:** All returned job arrays must map precisely to the standard schema: `title`, `company`, `location`, `is_remote`, `date_posted`, `interval`, `min_amount`, `max_amount`, `is_easy_apply`, `url`, `description`.
